# Singing-Style-transfer

## Requirement
- librosa==0.6.2
- numpy==1.14.3
- pyworld==0.2.4
- tensorflow==1.9.0
- python==3.6.5

## Usage
Before Training, you need to Prepare two Datasets and each must be located in "./data/train/A", "./data/train/B"
<pre><code>$ python train.py 
$ python test.py</code></pre>
